# base-resume
this is my resume for a front-end web development Job.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### 使用方法

在src目录下创建info.js 文件, 给定如下配置, 进行自定义内容编写: 
export default {
  headInfo: [],
  mainInfo: []
}
